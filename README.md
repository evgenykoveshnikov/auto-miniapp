# Мини авто приложение (Next.js)

## Установка и Запуск

### 1. Клонирование Репозитория

```bash
    git clone https://github.com/evgenykoveshnikov/auto-miniapp.git
    cd auto-miniapp
```

### 2. Установка зависимостей

```bash
    npm install
```

### Запуск проекта

1. В режиме разработке:
   ```bash
    npm run dev
   ```
2. В режиме producation:
   ```bash
    npm run build
    npm run start
   ```
